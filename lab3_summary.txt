Lab M3.03 - Pushing Your Code to GitHub Summary

1. What is SSH and why is it useful?

SSH stands for Secure Shell. It is a secure way for my computer to connect to GitHub.
Instead of typing my GitHub password every time, my computer uses an SSH key.
The private key stays on my laptop, and the public key is added to GitHub.

SSH is useful because it is secure, faster, and easier for pushing and pulling code.

2. What is a remote repository?

A remote repository is the online version of my local Git project.
In this lab, my remote repository is hosted on GitHub.

My local repository is on my laptop.
My remote repository is on GitHub.

3. What does git push do?

git push uploads my local commits from my computer to GitHub.
After pushing, my code becomes visible in my GitHub repository.
